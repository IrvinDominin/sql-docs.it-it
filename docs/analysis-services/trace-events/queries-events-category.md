# <a name="queries-events-category"></a>Categoria degli eventi di query
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
La categoria di eventi Eventi di query include le classi di eventi descritte nella tabella seguente.  
  
|Event Class|ID evento|Description|  
|-----------------|--------------|-----------------|  
|Query Begin|9|Raccoglie tutti gli eventi di inizio query dall'avvio della traccia.|  
|Query End|10|Raccoglie tutti gli eventi di fine query dall'avvio della traccia.|  
  
 Per informazioni sulle colonne associate a ogni classe di evento inclusa nella categoria Eventi di query, vedere [Colonne di dati degli eventi di query](../../analysis-services/trace-events/queries-events-data-columns.md).
