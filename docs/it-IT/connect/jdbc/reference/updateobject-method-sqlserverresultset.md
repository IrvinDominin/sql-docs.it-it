# <a name="updateobject-method-sqlserverresultset"></a>Metodo updateObject (SQLServerResultSet)

|Nome|Description|  
|----------|-----------------|  
|[updateObject (int, java.lang.Object)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object.md)|Aggiorna la colonna designata con un **oggetto** valore in base all'indice di colonna.|  
|[updateObject (int, java.lang.Object, int)](../../../connect/jdbc/reference/updateobject-method-int-java-lang-object-int.md)|Aggiorna la colonna designata con un **oggetto** valore base all'indice di colonna e una scala.|
