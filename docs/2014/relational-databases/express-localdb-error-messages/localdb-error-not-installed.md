# <a name="localdberrornotinstalled"></a>LOCALDB_ERROR_NOT_INSTALLED
    
## <a name="details"></a>Dettagli  
  
|||  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|278|  
|Origine evento|Runtime database locale di SQL Server 12.0|  
|Componente|API di Runtime database locale|  
|Testo del messaggio|**Nota:** il testo del messaggio è vuoto, perché questo messaggio indica che l'intera API LocalDB (inclusa la funzione FormatMessage che esegue il mapping di HRESULTS nel testo del messaggio) non è disponibile.|  
  
## <a name="explanation"></a>Spiegazione  
 Runtime database locale non installato nel computer.  
  
## <a name="user-action"></a>Azione dell'utente  
 Installare Runtime database locale, quindi ripetere l'operazione.
