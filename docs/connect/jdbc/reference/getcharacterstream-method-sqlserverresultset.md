# <a name="getcharacterstream-method-sqlserverresultset"></a>getCharacterStream (metodo) (SQLServerResultSet)

  Recupera il valore della colonna designata nella riga corrente di questo [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) oggetto come oggetto java.IO. Reader.
