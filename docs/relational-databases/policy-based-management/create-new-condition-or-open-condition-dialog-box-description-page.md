## <a name="options"></a>Opzioni  
 **Descrizione**  
 Consente di digitare una descrizione della condizione.  
  
 **Data creazione**  
 Data e ora di creazione della condizione.  
  
 **Creato da**  
 Account di accesso che ha creato la condizione.
