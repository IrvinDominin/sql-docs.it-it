# <a name="setbyte-method-sqlserverpreparedstatement"></a>setByte (metodo) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il parametro designato il dato **byte** valore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public final void setByte(int n,  
                          byte x)  
```  
  
#### <a name="parameters"></a>Parametri  
 *n*  
  
 Un **int** che indica il numero di parametro.  
  
 *x*  
  
 Oggetto **byte** valore.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo setByte viene specificato dal metodo setByte nell'interfaccia Java.SQL. PreparedStatement.
