# <a name="gettimestamp-method-datetimeoffset"></a>Metodo getTimestamp (DateTimeOffset)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce il punto nel tempo rappresentato da questo **DateTimeOffset** oggetto come un Java.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.sql.Timestamp getTimestamp()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Valore di tipo java.sql.Timestamp.
