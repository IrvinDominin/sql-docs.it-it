# <a name="msreportserverinstance-class"></a>classe MSReportServer_Instance

 Per un elenco di tutti i membri di questo tipo, vedere [Membri di MSReportServer_Instance](../../reporting-services/wmi-provider-library-reference/msreportserver-instance-members.md)
