# <a name="monitoring-python-in-sql-server"></a>Monitoraggio Python in SQL Server
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]

Supporto per Python è una nuova funzionalità di SQL Server, disponibile a partire da SQL Server 2017.

Il framework generale per la protezione dati e l'esecuzione dello script esterno è identico a quello di R. È possibile monitorare tutti gli script esterni, tra cui uno che esegue Python, con lo stesso framework che è stato specificato per l'esecuzione di script R in SQL Server 2016. Per altre informazioni, vedere [Monitoring R Solutions](../r/managing-and-monitoring-r-solutions.md).
