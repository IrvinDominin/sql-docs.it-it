# <a name="sinstance-class"></a>Classe SInstance
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Il [classe SInstance](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) rappresenta un'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Il [classe SInstance](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) può essere utilizzato per gestire protocolli e associate all'istanza di librerie di rete [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]. È inoltre possibile specificare un certificato di sicurezza per l'istanza di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
