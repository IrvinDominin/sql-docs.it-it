# <a name="overview-of-the-jdbc-driver"></a>Panoramica del driver JDBC
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  Microsoft JDBC Driver 4 (Anteprima) per SQL Server è un driver conforme a Java Database Connectivity (JDBC) 2017 di tipo 4.2 che offre potenti funzionalità di accesso ai dati per i database di SQL Server 2016 (Anteprima), SQL Server 2014, SQL Server 2012, SQL Server 2008 R2, SQL Server 2008 e Database SQL di Azure.  
  
 Negli argomenti di questa sezione viene fornita una panoramica generale del driver JDBC. Sono inclusi i requisiti di sistema necessari, le modalità di utilizzo del driver e la reperibilità di ulteriori informazioni.
