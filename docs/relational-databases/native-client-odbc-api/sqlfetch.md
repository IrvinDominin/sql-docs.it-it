# <a name="sqlfetch"></a>SQLFetch
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  SQLFetch supporta caratteristiche avanzate di data e ora.  
  
 Per informazioni su come vengono convertiti i valori di colonna di risultati dei tipi date/time, vedere [le conversioni da SQL a C](../../relational-databases/native-client-odbc-date-time/datetime-data-type-conversions-from-sql-to-c.md).  
  
## <a name="sqlfetch-support-for-large-clr-udts"></a>Supporto di SQLFetch per tipi CLR definiti dall'utente di grandi dimensioni  
 **SQLFetch** supporta grandi CLR tipi definiti dall'utente (UDT). Per altre informazioni, vedere [Large CLR User-Defined tipi &#40;ODBC&#41;](../../relational-databases/native-client/odbc/large-clr-user-defined-types-odbc.md).
