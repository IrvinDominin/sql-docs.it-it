  Utilizzare questa finestra di dialogo per selezionare uno schema di codifica della lingua e specificare i caratteri da inserire alla fine di ogni riga. Per accedere al **opzioni di salvataggio avanzate** finestra di dialogo il **File** menu, fare clic su **salvare** \<nome query >**SQL come** per Aprire la **Salva File con nome** finestra di dialogo, fare clic sulla freccia verso il basso nel **salvare** pulsante e quindi fare clic su **Salva con codifica**.
