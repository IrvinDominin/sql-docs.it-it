# <a name="table-valued-parameters-sql-server-native-client"></a>Parametri con valori di tabella (SQL Server Native Client)
  I parametri con valori di tabella sono stati introdotti in [!INCLUDE[ssKatmai](../../../includes/sskatmai-md.md)]e forniscono un modo efficiente per passare più righe di dati al server. I parametri con valori di tabella offrono una funzionalità simile alle matrici di parametri, ma offrono più flessibilità e maggiore integrazione con [!INCLUDE[tsql](../../../includes/tsql-md.md)]e spesso anche prestazioni migliori. I parametri con valori di tabella possono anche partecipare operazioni basate su set, mentre matrici di parametri.  
  
 Per informazioni sui parametri con valori di tabella e ODBC, vedere [Table-Valued Parameters &#40;ODBC&#41;](../../native-client-odbc-table-valued-parameters/table-valued-parameters-odbc.md).  
  
 Per informazioni sui parametri con valori di tabella e OLE DB, vedere [Table-Valued Parameters &#40;OLE DB&#41;](../../native-client-ole-db-table-valued-parameters/table-valued-parameters-ole-db.md).
