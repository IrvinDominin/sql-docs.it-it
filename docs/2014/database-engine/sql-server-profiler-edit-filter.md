# <a name="sql-server-profiler---edit-filter"></a>SQL Server Profiler - Modifica filtro
  Utilizzare la finestra di dialogo **Modifica filtro** per creare e modificare i filtri delle colonne di dati in una traccia. Fare clic sul nome di una colonna di dati nell'elenco per visualizzare nel riquadro adiacente i criteri di filtro disponibili per la colonna di dati. Immettere i criteri di filtro e fare clic su **OK** per applicarli alla colonna di dati selezionata. La presenza dell'icona del filtro a sinistra del nome della colonna di dati nell'elenco indica che la colonna dispone giÃ  di un filtro configurato.  
  
> [!NOTE]  
>  Per le colonne di dati di tipo stringa, i criteri di filtro saranno visualizzati come un valore di stringa LIKE o NOT LIKE.
