# <a name="protocolname-property-clientnetlibinfo-class"></a>Proprietà ProtocolName (classe ClientNetLibInfo)
  Ottiene il nome del protocollo associato alla libreria di rete del client.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto della [classe ClientNetLibInfo](clientnetlibinfo-class.md) che rappresenta le informazioni sulla libreria di rete del client.  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore string che specifica il nome del protocollo associato alla libreria di rete del client.
