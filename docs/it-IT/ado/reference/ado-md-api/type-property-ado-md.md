# <a name="type-property-ado-md"></a>Proprietà Type (ADO MD)
Indica il tipo dell'oggetto corrente [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Valori restituiti  
 Restituisce un [MemberTypeEnum](../../../ado/reference/ado-md-api/membertypeenum.md) valore ed è di sola lettura.  
  
## <a name="remarks"></a>Osservazioni  
 Questa proprietà è supportata solo in [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) oggetti appartenenti a un [livello](../../../ado/reference/ado-md-api/level-object-ado-md.md) oggetto. Si verifica un errore quando questa proprietà viene fatto riferimento dal **membro** oggetti appartenenti a un [posizione](../../../ado/reference/ado-md-api/position-object-ado-md.md) oggetto.
