# <a name="data-types-in-odbc"></a>Tipi di dati in ODBC
ODBC usa due set di tipi di dati: SQL tipi di dati e tipi di dati C. Tipi di dati SQL sono usati nell'origine dati, e tipi di dati C nel codice C nell'applicazione.
