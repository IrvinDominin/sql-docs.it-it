# <a name="expressions-dmx"></a>Espressioni (DMX)
[!INCLUDE[ssas-appliesto-sqlas](../includes/ssas-appliesto-sqlas.md)]

  In Data Mining di Extensions (DMX), un'espressione è una combinazione di identificatori, valori e operatori che [!INCLUDE[msCoName](../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] può valutare per ottenere un risultato.  
  
 Un'espressione DMX può essere semplice o complessa. Sono considerati semplici i tipi di espressioni seguenti:  
  
 Costante  
 Una costante è un simbolo che rappresenta un singolo valore specifico. Può essere una stringa oppure un valore numerico o di data. Per delimitare costanti di tipo string o date è necessario utilizzare le virgolette singole (').  
  
 Funzioni scalari  
 Una funzione scalare restituisce un singolo valore.  
  
 Funzioni non scalari  
 Una funzione non scalare restituisce una tabella.  
  
 Identificatori di oggetto  
 In DMX gli identificatori di oggetto sono considerati espressioni semplici.  
  
 Questi tipi di espressioni possono essere combinati tramite operatori per compilare espressioni complesse. Per altre informazioni sugli operatori, vedere [Data Mining Extensions &#40;DMX&#41; di riferimento agli operatori](../dmx/data-mining-extensions-dmx-operator-reference.md).
