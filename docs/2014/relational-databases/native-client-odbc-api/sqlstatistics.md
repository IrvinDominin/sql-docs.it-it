# <a name="sqlstatistics"></a>SQLStatistics
  **SQLStatistics** può essere eseguito su un cursore statico. Un tentativo di eseguire **SQLStatistics** su un cursore aggiornabile (dinamico o basati su keyset) restituirà SQL_SUCCESS_WITH_INFO a indicare che il tipo di cursore viene modificato.
