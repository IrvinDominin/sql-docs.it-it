# <a name="task-5-adding-data-flow-task"></a>Attività 5: Aggiunta dell'attività Flusso di dati
  In questa attività viene aggiunta un'Attività Flusso di dati al flusso di controllo del pacchetto SSIS.  
  
1.  Trascinamento della selezione **Data Flow Task** da **casella degli strumenti SSIS** per il **flusso di controllo** scheda nella finestra di progettazione SSIS. Se non viene visualizzato il **casella degli strumenti SSIS**, fare clic nel **flusso di controllo** scheda, fare clic su **SSIS** sulla barra dei menu, scegliere **casella degli strumenti SSIS**.  
  
     ![Flusso di dati di trascinamento attività dalla casella degli strumenti SSIS](../../2014/tutorials/media/et-addingdataflowtask.jpg "trascinare dati flusso attività dalla casella degli strumenti SSIS")  
  
2.  Fare doppio clic sui **Data Flow Task** nel **flusso di controllo** scheda e fare clic su **rinominare**.  
  
3.  Tipo di **Ricevi, Pulisci, corrispondenza e dati fornitore abbina** e premere **invio**.  
  
4.  Fare doppio clic sui **Data Flow Task** per passare al **del flusso di dati** scheda.
