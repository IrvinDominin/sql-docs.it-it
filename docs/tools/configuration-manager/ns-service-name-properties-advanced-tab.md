# <a name="nsltservice-namegt-properties-advanced-tab"></a>Proprietà NS$&lt;nome servizio&gt; (scheda Avanzate)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md-winonly](../../includes/appliesto-ss-xxxx-xxxx-xxx-md-winonly.md)]
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] viene implementato come servizio denominato nel formato **NS$***<nome_configurato_da_utente>*. Se sono state definite proprietà personalizzate, verranno visualizzate in questa scheda con i rispettivi valori. È possibile che questa scheda non contenga proprietà.
