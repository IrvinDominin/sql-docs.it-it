# <a name="complete---repair"></a>Operazione completata - Ripristino
  Utilizzare la pagina **Operazione completata - Ripristino** per verificare il completamento del ripristino.  
  
## <a name="options"></a>Opzioni  
 Questa finestra di dialogo include un collegamento al file di log del programma di installazione per questa operazione. Per continuare, fare clic su **Chiudi**.
