# <a name="isstatementpoolingenabled-method-sqlserverconnection"></a>isStatementPoolingEnabled metodo (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

 Restituisce se il pool di istruzione è attivato o meno per la connessione.

## <a name="syntax"></a>Sintassi  
  
```  
  
public boolean isStatementPoolingEnabled()  
```  

## <a name="return-value"></a>Valore restituito
 Oggetto **booleano** che contiene il flag che indica se il pool di istruzione è abilitato o meno.

## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
 
## <a name="remarks"></a>Osservazioni  
 Questo metodo è disponibile dal driver JDBC versione 6.4 e successivo.
