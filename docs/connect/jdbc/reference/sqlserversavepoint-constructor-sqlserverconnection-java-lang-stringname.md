# <a name="sqlserversavepoint-constructor-sqlserverconnection-javalangstringname"></a>Costruttore SQLServerSavepoint (SQLServerConnection, java.lang.StringName)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Inizializza una nuova istanza di [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) classe in base alla connessione specificata e il nome.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public SQLServerSavepoint(SQLServerConnection con,  
                          java.lang.String sName)  
```  
  
#### <a name="parameters"></a>Parametri  
 *con*  
  
 Oggetto [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md) oggetto.  
  
 *sName*  
  
 Oggetto **stringa** che contiene il nome del punto di salvataggio.
