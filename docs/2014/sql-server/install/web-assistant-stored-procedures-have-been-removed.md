# <a name="web-assistant-stored-procedures-have-been-removed"></a>Le stored procedure di Pubblicazione guidata sul Web sono state rimosse
  Preparazione aggiornamento: è stato rilevato l'utilizzo di stored procedure di Pubblicazione guidata sul Web.  
  
## <a name="component"></a>Componente  
 [!INCLUDE[ssDE](../../includes/ssde-md.md)]  
  
## <a name="description"></a>Description  
 Stored procedure per pubblicazione guidata sul Web **sp_makewebtask**, **sp_dropwebtask**, **sp_runwebtask**, e **sp_enumcodepages** state rimosso.  
  
## <a name="corrective-action"></a>Azione correttiva  
 È consigliabile utilizzare [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] in alternativa.
