# <a name="errors-ado"></a>Errori (ADO)
Qualsiasi operazione che interessa oggetti ADO è possibile generare uno o più errori del provider. Come si verifica ogni errore, una o più **errore** gli oggetti vengono inseriti nel **errori** raccolta del **connessione** oggetto. Per informazioni dettagliate sulla gestione degli avvisi ed errori nell'applicazione ADO, vedere [Error Handling](../../../ado/guide/data/error-handling.md).  
  
 Errori dell'applicazione possono essere generati da un meccanismo separato. Ad esempio, in Visual Basic, il **Err** oggetto conterrà errori a livello di applicazione.
