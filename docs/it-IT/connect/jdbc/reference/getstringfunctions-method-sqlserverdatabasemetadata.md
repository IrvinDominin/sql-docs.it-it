# <a name="getstringfunctions-method-sqlserverdatabasemetadata"></a>Metodo getStringFunctions (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera un elenco delimitato da virgole di **stringa** funzioni che sono disponibili con il database.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.lang.String getStringFunctions()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Oggetto **stringa** che contiene il **stringa** funzioni.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo getStringFunctions viene specificato dal metodo getStringFunctions nell'interfaccia DatabaseMetaData.
