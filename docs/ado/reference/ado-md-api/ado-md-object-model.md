# <a name="ado-md-object-model"></a>Modello a oggetti ADO MD
Questo argomento viene descritto come gli oggetti vengono rappresentati e correlati in ADO MD.  
  
 ![Modello a oggetti ADO MD](../../../ado/reference/ado-md-api/media/ado_md_object_model.gif "ADO_MD_object_model")  
  
 Il [asse](../../../ado/reference/ado-md-api/axis-object-ado-md.md) e [cella](../../../ado/reference/ado-md-api/cell-object-ado-md.md) ogni gli oggetti hanno una [posizioni](../../../ado/reference/ado-md-api/positions-collection-ado-md.md) insieme.  
  
 Il [livello](../../../ado/reference/ado-md-api/level-object-ado-md.md) e [posizione](../../../ado/reference/ado-md-api/position-object-ado-md.md) ogni gli oggetti hanno una [membri](../../../ado/reference/ado-md-api/members-collection-ado-md.md) insieme.  
  
 Il [asse](../../../ado/reference/ado-md-api/axis-object-ado-md.md), [cella](../../../ado/reference/ado-md-api/cell-object-ado-md.md), [set di celle](../../../ado/reference/ado-md-api/cellset-object-ado-md.md), [CubeDef](../../../ado/reference/ado-md-api/cubedef-object-ado-md.md), [dimensione](../../../ado/reference/ado-md-api/dimension-object-ado-md.md), [gerarchia](../../../ado/reference/ado-md-api/hierarchy-object-ado-md.md), [Livello](../../../ado/reference/ado-md-api/level-object-ado-md.md), e [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) ogni oggetto dispone di un oggetto ADO standard [proprietà](../../../ado/reference/ado-api/properties-collection-ado.md) insieme.
