# <a name="installer-dll"></a>Programma di installazione di DLL
> [!NOTE]  
>  A partire da Windows XP e Windows Server 2003, ODBC è incluso nel sistema operativo Windows. Solo nelle versioni precedenti di Windows è necessario installare ODBC in modo esplicito.  
  
 Il programma di installazione DLL contiene funzioni per installare e rimuovere componenti ODBC, gestire le informazioni del Registro di sistema su tali componenti e gestire le informazioni del Registro di sistema relative origini dati. Scritto da Microsoft e può essere ridistribuito. Per una descrizione completa delle funzioni di DLL del programma di installazione, vedere [Installer DLL API Reference](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
