# <a name="propertyindex-property-sqlserviceadvancedproperty-class"></a>Proprietà PropertyIndex (classe SqlServiceAdvancedProperty)
  Ottiene o imposta l'indice di proprietà che specifica la posizione di una proprietà avanzata in una matrice di proprietà avanzate che appartengono a un servizio a cui si fa riferimento.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.PropertyIndex [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto della [classe SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) che rappresenta una proprietà avanzata.  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore `uint32` che specifica la posizione della proprietà avanzata nella matrice di proprietà avanzate che appartengono al servizio a cui si fa riferimento.
