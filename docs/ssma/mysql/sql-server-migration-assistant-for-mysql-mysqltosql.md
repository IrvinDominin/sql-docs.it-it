# <a name="sql-server-migration-assistant-for-mysql-mysqltosql"></a>SQL Server Migration Assistant per MySQL (MySQLToSQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] Migration Assistant (SSMA) per MySQL è uno strumento per la migrazione di database MySQL al [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 2005 / [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 2008 / [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 2012 / [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 2014 / [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 2016 / [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] 2017 in Windows e Linux (anteprima) / Azure [!INCLUDE[msCoName](../../includes/msconame_md.md)] DB. SSMA per MySQL converte gli oggetti di database MySQL in oggetti di database di SQL Server, questi oggetti vengono creati in [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)], e quindi viene eseguita la migrazione dei dati da MySQL a [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)].  
  
Questa documentazione viene presentato SSMA per MySQL e vengono fornite istruzioni dettagliate per la migrazione di database MySQL a [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]. Nella tabella seguente mostra gli argomenti per ulteriori informazioni:
