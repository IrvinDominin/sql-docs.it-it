# <a name="getboolean-method-sqlservercallablestatement"></a>Metodo getBoolean (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **booleano** valore.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getBoolean (int)](../../../connect/jdbc/reference/getboolean-method-int.md)|Recupera il valore del parametro designato come un **booleano** valore in base all'indice del parametro.|  
|[getBoolean (lang)](../../../connect/jdbc/reference/getboolean-method-java-lang-string.md)|Recupera il valore del parametro designato come un **booleano** valore in base al nome del parametro.|
