# <a name="configuration-components"></a>Componenti di configurazione
> [!NOTE]  
>  A partire da Windows XP e Windows Server 2003, ODBC è incluso nel sistema operativo Windows. Solo nelle versioni precedenti di Windows è necessario installare ODBC in modo esplicito.  
  
 Origini dati siano configurate dal programma di installazione DLL, che nel driver chiamate di attivazione del programma di installazione DLL e file DLL di installazione di funzione di conversione, se necessario. Il programma di installazione DLL è sia richiamato direttamente dal Pannello di controllo o caricata e chiamata da un altro programma, noto come il *programma di amministrazione*.
