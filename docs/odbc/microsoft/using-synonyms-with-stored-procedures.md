# <a name="using-synonyms-with-stored-procedures"></a>Uso dei sinonimi con le stored procedure
> [!IMPORTANT]  
>  Questa funzionalità verrà rimossa in una versione futura di Windows. Evitare di usare questa funzionalità in un nuovo progetto di sviluppo e prevedere interventi di modifica nelle applicazioni in cui è attualmente implementata. In alternativa, usare il driver ODBC fornito da Oracle.  
  
 Microsoft ODBC Driver per le versioni 2.0 e 2.5 di Oracle non supportano i sinonimi Oracle chiamante le stored procedure. Sinonimi funzionano come previsto quando usato con altri oggetti di database Oracle, ad esempio le tabelle.
