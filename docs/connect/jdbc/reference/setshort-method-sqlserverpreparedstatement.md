# <a name="setshort-method-sqlserverpreparedstatement"></a>setShort (metodo) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il parametro designato il dato **breve** valore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public final void setShort(int index,  
                           short x)  
```  
  
#### <a name="parameters"></a>Parametri  
 *index*  
  
 Un **int** che indica il numero di parametro.  
  
 *x*  
  
 Oggetto **breve** valore.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo setShort viene specificato dal metodo setShort nell'interfaccia Java.SQL. PreparedStatement.
