# <a name="leveldepth-property-ado-md"></a>Proprietà LevelDepth (ADO MD)
Indica il numero di livelli tra la radice della gerarchia e un [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Valori restituiti  
 Restituisce un **lungo** intero ed è di sola lettura.  
  
## <a name="remarks"></a>Osservazioni  
 Utilizzare il **LevelDepth** proprietà per determinare la distanza tra il [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md)oggetto dal livello radice della gerarchia. Il **LevelDepth**di un membro alla radice level è 0. Corrisponde alla [profondità](../../../ado/reference/ado-md-api/depth-property-ado-md.md) proprietà di un [livello](../../../ado/reference/ado-md-api/level-object-ado-md.md) oggetto.
