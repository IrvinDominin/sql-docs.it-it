# <a name="mssqlrepl-2147200941"></a>MSSQL_REPL-2147200941
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## <a name="message-details"></a>Dettagli messaggio  
  
|||  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|-2147200941|  
|Origine evento|MSSQLServer|  
|Nome simbolico||  
|Testo del messaggio|Durante il processo di merge è stata rilevata una mancata corrispondenza in fase di valutazione dell'espressione di convalida delle partizioni del Sottoscrittore. Per risolvere il problema, reinizializzare la sottoscrizione.|  
  
## <a name="explanation"></a>Spiegazione  
 Durante il processo di merge è stata rilevata una mancata corrispondenza in fase di valutazione dell'espressione di convalida delle partizioni del Sottoscrittore.  
  
## <a name="user-action"></a>Azione dell'utente  
 Reinizializzare la sottoscrizione.
