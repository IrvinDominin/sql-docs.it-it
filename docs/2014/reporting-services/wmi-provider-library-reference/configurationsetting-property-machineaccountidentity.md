# <a name="machineaccountidentity-property-wmi"></a>Proprietà MachineAccountIdentity (WMI)
  Ottiene l'identità dell'account del computer in cui è installato il server di report.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>Valori della proprietà  
 Oggetto `String` valore che contiene l'identità dell'account macchina del computer in cui è installato il server di report.
