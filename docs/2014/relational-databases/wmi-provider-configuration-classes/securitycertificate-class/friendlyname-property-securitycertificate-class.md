# <a name="friendlyname-property-securitycertificate-class"></a>Proprietà FriendlyName (classe SecurityCertificate)
  Ottiene o imposta un nome descrittivo alternativo per il certificato di sicurezza.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.FriendlyName [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto della [classe SecurityCertificate](securitycertificate-class.md) che rappresenta un certificato di sicurezza.  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore string che specifica un nome descrittivo alternativo per il certificato di sicurezza.
