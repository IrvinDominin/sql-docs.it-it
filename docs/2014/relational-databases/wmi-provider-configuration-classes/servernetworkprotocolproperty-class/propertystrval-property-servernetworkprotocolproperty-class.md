# <a name="propertystrval-property-servernetworkprotocolproperty-class"></a>Proprietà PropertyStrVal (classe ServerNetworkProtocolProperty)
  Ottiene il valore string della proprietà a cui si fa riferimento.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.PropertyStrVal [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto della [classe ServerNetworkProtocolProperty](servernetworkprotocolproperty-class.md) che rappresenta un attributo del protocollo di rete nell'istanza di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore string che specifica il valore della proprietà.
