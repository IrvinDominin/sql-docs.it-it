# <a name="getref-method-sqlservercallablestatement"></a>Metodo getRef (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come oggetto di riferimento nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getRef (int)](../../../connect/jdbc/reference/getref-method-int.md)|Recupera il valore del parametro designato come oggetto nel linguaggio di programmazione Java specificato all'indice del parametro Ref.|  
|[getRef (lang)](../../../connect/jdbc/reference/getref-method-java-lang-string.md)|Recupera il valore del parametro designato come oggetto nel linguaggio di programmazione Java specificato il nome del parametro Ref.|
