# <a name="deploying-an-exception-message-box-application"></a>Distribuzione di un'applicazione finestra di messaggio eccezione
  La finestra di messaggio eccezione viene installata da [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] ed è supportato per l'utilizzo nelle applicazioni Windows personalizzate per migliorare la gestione delle eccezioni. Poiché la finestra di messaggio eccezione viene installata da tutte le edizioni di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] eccetto [!INCLUDE[ssEW](../../includes/ssew-md.md)], è possibile utilizzare senza alcuna configurazione aggiuntiva in qualsiasi computer in cui [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] componenti client, incluso il SDK, sono stati installati.
