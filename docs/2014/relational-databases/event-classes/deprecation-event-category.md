# <a name="deprecation-event-category"></a>Deprecation - categoria di eventi
  La categoria di eventi **Deprecation** contiene eventi generali relativi a elementi deprecati.

|[Classe di evento Deprecation Final Support](deprecation-final-support-event-class.md)|Indica che una caratteristica verrà rimossa a partire dalla prossima versione principale di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].|
