# <a name="getbyte-method-sqlservercallablestatement"></a>Metodo getByte (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **byte** valore.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getByte (int)](../../../connect/jdbc/reference/getbyte-method-int.md)|Recupera il valore del parametro designato come un **byte** valore in base all'indice del parametro.|  
|[getByte (lang)](../../../connect/jdbc/reference/getbyte-method-java-lang-string.md)|Recupera il valore del parametro designato come un **byte** valore in base al nome del parametro.|
