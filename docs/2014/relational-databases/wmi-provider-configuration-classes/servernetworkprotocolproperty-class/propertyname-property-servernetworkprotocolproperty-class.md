# <a name="propertyname-property-servernetworkprotocolproperty-class"></a>Proprietà PropertyName (classe ServerNetworkProtocolProperty)
  Ottiene il nome della proprietà corrente a cui si fa riferimento.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 A [classe ServerNetworkProtocolProperty](servernetworkprotocolproperty-class.md) che rappresenta un attributo del protocollo di rete nell'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore string che specifica il nome della proprietà corrente.
