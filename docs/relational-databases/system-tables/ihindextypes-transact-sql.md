# <a name="ihindextypes-transact-sql"></a>IHindextypes (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Il **IHindextypes** tabella di sistema contiene una riga per ogni tipo di indice non SQL Server per SQL Server di pubblicazione non è supportato. Questa tabella è archiviata nel database di distribuzione.  
  
|Nome colonna|Tipo di dati|Description|  
|-----------------|---------------|-----------------|  
|**tipo**|**nvarchar(255)**|Il nome di un tipo di indice non SQL Server supportato.|
