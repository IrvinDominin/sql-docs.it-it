# <a name="discard-changes-made-to-diagrams-visual-database-tools"></a>Annullare le modifiche apportate ai diagrammi (Visual Database Tools)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
È possibile eliminare le modifiche al diagramma di database che non si desidera salvare nel database.  
  
### <a name="to-discard-pending-modifications"></a>Per eliminare le modifiche in sospeso  
  
1.  Chiudere la finestra del diagramma di database o la finestra di Progettazione tabelle.  
  
2.  Verrà visualizzato un messaggio in cui viene chiesto se si desidera salvare le modifiche. Scegliere **No**.  
  
    > [!CAUTION]  
    > Se sono aperte altre finestre di diagrammi di database o di Progettazione tabelle associate allo stesso database, chiuderle senza salvare le modifiche.
