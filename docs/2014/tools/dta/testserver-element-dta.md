# <a name="testserver-element-dta"></a>Elemento TestServer (DTA)

## <a name="syntax"></a>Sintassi  
  
```  
  
<TuningOptions>  
  <TestServer>...</TestServer>  
   ...code removed here...  
</TuningOptions>  
```
