# <a name="ordered-and-unordered-expressions-xquery"></a>Espressioni ordinate e non ordinate (XQuery)
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  Per impostazione predefinita, la modalità di ordinamento per tutte le operazioni [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] viene **ordinati**. Pertanto, le sequenze di nodi restituite dalle espressioni di percorso e dalle espressioni FLWOR, senza la **ordinare in base a** clausola, sono nell'ordine del documento.  
  
 Aggiuntiva **ordinati** e **ordinato** sintassi descritta nella specifica XQuery non è supportata.
