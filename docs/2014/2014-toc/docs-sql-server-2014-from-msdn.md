# Documentazione per SQL Server 2014 è lo spostamento in documenti

In questo articolo viene descritto lo spostamento imminente di articoli di documentazione di Microsoft SQL Server 2014. In questo articolo viene scritto in ritardo 2018 aprile e lo spostamento verificherà prima 2018 luglio.

## Collegamenti alla documentazione di SQL Server 2014 in MSDN

I seguenti collegamenti rimandano la documentazione di SQL Server 2014 che Microsoft archiviati nel relativo *MSDN* sito Web. Prima di luglio 2018 Microsoft sposterà gli articoli di documentazione per il nuovo *Docs* sito Web:

- *SQL Server 2014*
    - [https://msdn.microsoft.com/library/dn197878(v=sql.10).aspx](https://msdn.microsoft.com/library/dn197878(v=sql.10).aspx)

- *Documentazione online per SQL Server 2014*
    - [https://msdn.microsoft.com/library/ms130214(v=sql.120).aspx](https://msdn.microsoft.com/library/ms130214(v=sql.120).aspx)

MSDN è un initialism per *Microsoft Developer Network*.


## Redirector

Gli indirizzi HTTP MSDN obsoleti continueranno a funzionare per almeno due anni dopo lo spostamento. Redirector intercettano le richieste MSDN e li inoltrano con indirizzo di posta nuovo corrispondente su documenti. Tuttavia, dopo lo spostamento gli elementi seguenti sarà true:

- La procedura migliore funzionare con i nuovi indirizzi HTTP che passare direttamente alla Docs.
- Verrà avviato il crawler Bing e Google cancellazione dai listini e ignorare gli indirizzi MSDN precedenti.


## Dopo lo spostamento

Dopo lo spostamento *Docs*, l'indirizzo dell'articolo di documentazione principale di SQL Server 2014 sarà:

- [https://docs.microsoft.com/sql/?view=sql-server-2014](https://docs.microsoft.com/sql/?view=sql-server-2014)
