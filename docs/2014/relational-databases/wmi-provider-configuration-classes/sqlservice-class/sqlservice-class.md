# <a name="sqlservice-class"></a>Classe SqlService
  La [classe SqlService](sqlservice-class.md) rappresenta un'istanza di un servizio [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] .  
  
 La [classe SqlService](sqlservice-class.md) viene utilizzata per gestire a livello di programmazione il servizio [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] . Con la [classe SqlService](sqlservice-class.md) è possibile effettuare le attività seguenti:  
  
-   Connettersi al servizio.  
  
-   Modificare l'account e la password utilizzati per connettersi al servizio.  
  
-   Avviare, arrestare, sospendere e riprendere il servizio.  
  
-   Verificare le proprietà del servizio.
