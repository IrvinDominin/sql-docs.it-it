# <a name="cube-element-olapinfo-xmla"></a>Elemento Cube (OlapInfo) (XMLA)
  Contiene informazioni su un cubo per l'elemento padre [CubeInfo](cubeinfo-element-xmla.md) elemento.  
  
## <a name="syntax"></a>Sintassi  
  
```xml  
  
<CubeInfo>  
   <Cube>  
      <CubeName>...</CubeName>  
      <LastDataUpdate>...</LastDataUpdate>  
      <LastSchemaUpdate>...</LastSchemaUpdate>  
   </Cube>  
   ...  
</CubeInfo>  
```  
  
## <a name="element-characteristics"></a>Caratteristiche elemento  
  
|Caratteristica|Description|  
|--------------------|-----------------|  
|Tipo di dati e lunghezza|None|  
|Valore predefinito|None|  
|Cardinalità|1-1: elemento obbligatorio visualizzato una sola volta.|

|------------------|-------------|  
|Elementi padre|[CubeInfo](cubeinfo-element-xmla.md)|  
|Elementi figlio|[CubeName](name-element-xmla.md), [LastDataUpdate](../xml-elements-commands/update-element-xmla.md), [LastSchemaUpdate](lastschemaupdate-element-xmla.md)|
