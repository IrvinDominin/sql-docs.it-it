# <a name="getdouble-method-sqlservercallablestatement"></a>Metodo getDouble (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **doppie** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getDouble (int)](../../../connect/jdbc/reference/getdouble-method-int.md)|Recupera il valore del parametro designato come un **doppie** in base all'indice del parametro di linguaggio di programmazione Java.|  
|[getDouble (lang)](../../../connect/jdbc/reference/getdouble-method-java-lang-string.md)|Recupera il valore del parametro designato come un **doppie** nel linguaggio di programmazione Java specificato il nome del parametro.|
