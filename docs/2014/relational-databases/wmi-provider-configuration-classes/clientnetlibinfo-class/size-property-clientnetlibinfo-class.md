# <a name="size-property-clientnetlibinfo-class"></a>Proprietà Size (classe ClientNetLibInfo)
  Ottiene le dimensioni in kilobyte della libreria di rete del client.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
object  
.Size [= value]  
```  
  
## <a name="parts"></a>Parti  
 *object*  
 Oggetto della [classe ClientNetLibInfo](clientnetlibinfo-class.md) che rappresenta le informazioni sulla libreria di rete del client.  
  
## <a name="property-valuereturn-value"></a>Valore proprietà/Valore restituito  
 Valore uint32 che specifica le dimensioni in KB della libreria di rete del client.
