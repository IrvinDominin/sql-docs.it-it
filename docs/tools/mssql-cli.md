# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>strumento da riga di comando di query MSSQL-cli per SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

MSSQL-cli è uno strumento da riga di comando interattivo per le query di SQL Server, installarlo in Windows, Mac OS o Linux.

## <a name="install-mssql-cli"></a>Installare mssql cli

Per istruzioni dettagliate sull'installazione, vedere il [Guida all'installazione](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md), o se si conosce pip, installare eseguendo il comando seguente:

```$ pip install mssql-cli```
