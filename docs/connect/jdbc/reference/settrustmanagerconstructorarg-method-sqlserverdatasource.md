# <a name="settrustmanagerconstructorarg-method-sqlserverdatasource"></a>Metodo setTrustManagerConstructorArg (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il valore di stringa della proprietà di connessione TrustManagerConstructorArg.
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public void setTrustManagerConstructorArg(java.lang.String trustManagerClass)  
```  
  
#### <a name="parameters"></a>Parametri  
 *Le proprietà di connessione*  
  
 Oggetto **stringa** che contiene il nome completo della classe di un javax.net.ssl.TrustManager personalizzato.
