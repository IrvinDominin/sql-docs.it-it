# <a name="cluster-security-policy"></a>Criteri di sicurezza cluster
  Utilizzare la pagina Criteri di sicurezza cluster per configurare i criteri di sicurezza per l'istanza del cluster di failover.  
  
## <a name="options"></a>Opzioni  
 Specificare gruppi a livello di dominio globale o locale per i servizi di cluster. Tutte le autorizzazioni per le risorse sono controllate da gruppi a livello di dominio che includono gli account del servizio [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] come membri del gruppo.
