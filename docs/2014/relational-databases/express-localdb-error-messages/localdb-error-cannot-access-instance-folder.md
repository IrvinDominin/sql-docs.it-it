# <a name="localdberrorcannotaccessinstancefolder"></a>LOCALDB_ERROR_CANNOT_ACCESS_INSTANCE_FOLDER
    
## <a name="details"></a>Dettagli  
  
|||  
|-|-|  
|Nome prodotto|SQL Server|  
|ID evento|261|  
|Origine evento|Runtime database locale di SQL Server 12.0|  
|Componente|API di Runtime database locale|  
|Testo del messaggio|Impossibile accedere alla cartella istanza Database locale: %%LOCALAPPDATA%%\Microsoft\Microsoft SQL Server locale DB\Instances\\< nome istanza\>.|  
  
## <a name="explanation"></a>Spiegazione  
 Impossibile accedere alla cartella di un'istanza.
