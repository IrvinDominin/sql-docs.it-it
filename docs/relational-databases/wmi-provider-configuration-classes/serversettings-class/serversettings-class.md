Il [classe ServerSettings](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md) rappresenta le impostazioni del server in un'istanza di [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 Il [classe ServerSettings](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md) consente di eseguire le attivit√† seguenti:

-   Impostare i valori predefiniti per l'istanza di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
-   Impostare il certificato di sicurezza per l'istanza di [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
