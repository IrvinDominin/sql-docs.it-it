# <a name="sql-server-native-client-ole-db-reference"></a>Guida di riferimento di SQL Server Native Client (OLE DB)
  Il [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] provider OLE DB Native Client espone interfacce ai consumer che richiedono l'accesso ai dati in uno o più computer che eseguono [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Il provider OLE DB di [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Native Client è un provider compatibile con OLE DB versione 2.0.  
  
 Ciò [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] programmazione riferimento del provider OLE DB Native Client non è documentate tutte le interfacce OLE DB e i metodi, solo le interfacce e metodi che possono presentare un comportamento specifico del provider quando si utilizza il [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] OLE DB Native Client provider.  
  
 Per una descrizione completa delle interfacce OLE DB principale e dei metodi, vedere la [riferimento per programmatori OLE DB](http://go.microsoft.com/fwlink/?LinkId=45232).
