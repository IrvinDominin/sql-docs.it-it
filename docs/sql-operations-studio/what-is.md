# <a name="what-is-includename-sosincludesname-sosmd"></a>Che cos'è [!INCLUDE[name-sos](../includes/name-sos.md)]?

[!INCLUDE[name-sos](../includes/name-sos-short.md)]è uno strumento gratuito che viene eseguito in Windows, macOS e Linux, per la gestione di SQL Server, Database SQL di Azure e Azure SQL Data Warehouse; punto in cui vengono eseguiti.

**[Scaricare e installare[!INCLUDE[name-sos](../includes/name-sos-short.md)]](download.md)**


## <a name="transact-sql-t-sql-code-editor-with-intellisense"></a>Editor di codice Transact-SQL (T-SQL) con IntelliSense

[!INCLUDE[name-sos](../includes/name-sos-short.md)]offre una moderna, con stato attivo della tastiera T-SQL codifica che semplifica le attività quotidiane con funzionalità incorporate, ad esempio più finestre scheda, un editor T-SQL avanzato, IntelliSense, completamento (parola chiave), frammenti di codice, spostamento di codice e controllo del codice sorgente integrazione (Git). Esecuzione di query T-SQL su richiesta, visualizzare e salvare i risultati come testo, JSON o Excel. Modificare i dati, organizzare le connessioni di database Preferiti e sfogliare oggetti di database in un oggetto familiarità esperienza di esplorazione. Per informazioni su come utilizzare l'editor T-SQL, vedere [utilizzare l'editor T-SQL per creare oggetti di database](tutorial-sql-editor.md).

## <a name="smart-t-sql-code-snippets"></a>Smart frammenti di codice T-SQL

Frammenti di codice T-SQL generano la sintassi T-SQL appropriata per creare database, tabelle, viste, stored procedure, gli utenti, account di accesso, ruoli, e così via e per aggiornare gli oggetti di database esistente. Utilizzare smart i frammenti di codice per creare rapidamente copie del database di sviluppo o a scopo di test e di generare ed eseguire Creazione e l'inserimento di script.

[!INCLUDE[name-sos](../includes/name-sos-short.md)]fornisce inoltre funzionalità per creare frammenti di codice T-SQL personalizzati. Per ulteriori informazioni, vedere [creare e usare i frammenti di codice](code-snippets.md).


## <a name="customizable-server-and-database-dashboards"></a>Personalizzabile dei Server e i dashboard di Database

Creare dashboard personalizzabile avanzati per monitorare e risolvere rapidamente i colli di bottiglia delle prestazioni del database. Per informazioni sulla visione widget e dashboard di database (e server), vedere [gestire server e i database con i widget insight](insight-widgets.md).

## <a name="connection-management-server-groups"></a>Gestione delle connessioni (gruppi di server)

Gruppi di server consentono di organizzare le informazioni di connessione per i server e si lavora con i database. Per informazioni dettagliate, vedere [gruppi di Server](server-groups.md).

## <a name="integrated-terminal"></a>Terminal integrata

Utilizzare gli strumenti da riga di comando Preferiti (ad esempio, barra rovesciata, PowerShell, sqlcmd e bcp e ssh) nella finestra Terminal integrata direttamente il [!INCLUDE[name-sos](../includes/name-sos-short.md)] interfaccia utente. Per ulteriori informazioni su terminal integrato, vedere [integrata terminal] (terminal.md integrato).
