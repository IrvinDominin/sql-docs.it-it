# <a name="getfloat-method-sqlservercallablestatement"></a>Metodo getFloat (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Recupera il valore del parametro designato come un **float** nel linguaggio di programmazione Java.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[getFloat (int)](../../../connect/jdbc/reference/getfloat-method-int.md)|Recupera il valore del parametro designato come un **float** in base all'indice del parametro di linguaggio di programmazione Java.|  
|[getFloat (lang)](../../../connect/jdbc/reference/getfloat-method-java-lang-string.md)|Recupera il valore del parametro designato come un **float** nel linguaggio di programmazione Java specificato il nome del parametro.|
