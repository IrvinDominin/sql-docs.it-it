# <a name="connect-to-database-dialog-box"></a>Finestra di dialogo Connetti al database
  Utilizzare la finestra di dialogo **Connetti al database** per connettersi a un database di [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] .  
  
## <a name="options"></a>Opzioni  
  
|Nome|Definizione|  
|----------|----------------|  
|**Connettersi al database esistente**|Selezionare questa opzione per connettersi a un database esistente. Nell'elenco di database esistenti fare clic su quello a cui si desidera effettuare la connessione.|  
|**Crea nuovo database**|Selezionare questa opzione per creare un nuovo database.|  
|**Aggiungere alla soluzione**|Selezionare questa opzione per aggiungere il database alla soluzione corrente.|  
|**Chiudi soluzione**|Selezionare questa opzione per aggiungere il database a una nuova soluzione.|
