# <a name="first-method-sqlserverresultset"></a>primo metodo (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Sposta il cursore sulla prima riga di questo [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) oggetto.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public boolean first()  
```  
  
## <a name="return-value"></a>Valore restituito  
 **true** se il cursore viene spostato nella prima riga. In caso contrario, **false**.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo prima viene specificato dal primo metodo nell'interfaccia Java.SQL. ResultSet.
