# <a name="open-a-trace-file-sql-server-profiler"></a>Aprire un file di traccia (SQL Server Profiler)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  In questo argomento viene descritta la procedura per l'apertura di un file di traccia tramite [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)].  
  
### <a name="to-open-a-trace-file"></a>Per aprire un file di traccia  
  
1.  Scegliere **Apri** dal menu **File**e quindi fare clic su **File di traccia**.  
  
2.  Nella finestra di dialogo **Apri file** selezionare il file di traccia che si desidera aprire.
