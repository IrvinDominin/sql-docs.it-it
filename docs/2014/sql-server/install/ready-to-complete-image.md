# <a name="ready-to-complete-image"></a>Inizio completamento immagine
  La pagina **Inizio completamento immagine** fornisce un riepilogo delle impostazioni di configurazione applicate alla configurazione dell'istanza predisposta.  
  
## <a name="options"></a>Opzioni  
 Nella pagina sono disponibili le opzioni seguenti:  
  
-   Per continuare l'installazione, fare clic su **Installa**.  
  
-   Per modificare le impostazioni di installazione, fare clic su **Indietro**.  
  
-   Per uscire dal programma di installazione, fare clic su **Annulla**.
