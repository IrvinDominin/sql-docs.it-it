# <a name="ihconstrainttypes-transact-sql"></a>IHconstrainttypes (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Il **IHconstrainttypes** tabella di sistema contiene una riga per ogni tipo di vincolo non SQL Server per SQL Server di pubblicazione non è supportato. Questa tabella è archiviata nel database di distribuzione.  
  
|Nome colonna|Tipo di dati|Description|  
|-----------------|---------------|-----------------|  
|**type**|**nvarchar(255)**|Nome di un tipo di vincolo non SQL Server supportato.|
