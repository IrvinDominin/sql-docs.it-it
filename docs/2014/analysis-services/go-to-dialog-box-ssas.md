# <a name="go-to-dialog-box-ssas"></a>Finestra di dialogo Vai a (SSAS)
  La finestra di dialogo **Vai a** consente di passare a una colonna specifica in una tabella.  
  
## <a name="uielement-list"></a>Elenco degli elementi di interfaccia  
  
|||  
|-|-|  
|Nome|Definizione|  
|**Vai a**|Selezionare una colonna dall'elenco **Vai a** , quindi fare clic su **OK**.|
