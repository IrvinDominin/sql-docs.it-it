# <a name="setapplicationintent-method-sqlserverdatasource"></a>Metodo setApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il valore della **applicationIntent** proprietà di connessione.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>Parametri  
 *applicationIntent*  
  
 Il nuovo valore della **applicationIntent** proprietà di connessione.  
  
## <a name="remarks"></a>Osservazioni  
 Per ulteriori informazioni sul **applicationIntent** proprietà di connessione, vedere [impostando le proprietà di connessione](../../../connect/jdbc/setting-the-connection-properties.md).
