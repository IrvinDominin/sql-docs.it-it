# <a name="setsavepoint-method-sqlserverconnection"></a>setSavepoint (metodo) (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Crea un punto di salvataggio senza nome nella transazione corrente e restituisce il nuovo [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) oggetto che lo rappresenta.  
  
## <a name="overload-list"></a>Elenco degli overload  
  
|Nome|Description|  
|----------|-----------------|  
|[setSavepoint)](../../../connect/jdbc/reference/setsavepoint-method.md)|Crea un punto di salvataggio senza nome nella transazione corrente e restituisce il nuovo [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) oggetto che lo rappresenta.|  
|[setSavepoint (lang)](../../../connect/jdbc/reference/setsavepoint-method-java-lang-string.md)|Crea un punto di salvataggio con il nome specificato nella transazione corrente e restituisce il nuovo [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) oggetto che lo rappresenta.|
