# <a name="parentsameasprev-property-ado-md"></a>Proprietà ParentSameAsPrev (ADO MD)
Indica se l'elemento padre di questa posizione [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) corrisponde a quello dell'elemento padre del membro immediatamente precedente.  
  
## <a name="return-values"></a>Valori restituiti  
 Restituisce un **booleano** valore ed è di sola lettura.  
  
## <a name="remarks"></a>Remarks  
 Questa proprietà è supportata solo in [membro](../../../ado/reference/ado-md-api/member-object-ado-md.md) oggetti appartenenti a un [posizione](../../../ado/reference/ado-md-api/position-object-ado-md.md) oggetto. Si verifica un errore quando questa proprietà viene fatto riferimento dal **membro** oggetti appartenenti a un [livello](../../../ado/reference/ado-md-api/level-object-ado-md.md) oggetto.
