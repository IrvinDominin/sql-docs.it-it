# <a name="setboolean-method-sqlserverpreparedstatement"></a>setBoolean (metodo) (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Imposta il parametro designato il dato **booleano** valore.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public final void setBoolean(int n,  
                             boolean x)  
```  
  
#### <a name="parameters"></a>Parametri  
 *n*  
  
 Un **int** che indica il numero di parametro.  
  
 *x*  
  
 Oggetto **booleano** valore, **true** o **false**.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo setboolean viene specificato dal metodo setboolean nell'interfaccia Java.SQL. PreparedStatement.
