# <a name="show-advanced-options-server-configuration-option"></a>Opzione di configurazione del server show advanced options
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]

  L'opzione **show advanced options** consente di visualizzare le opzioni avanzate della stored procedure di sistema **sp_configure** . Se si imposta **show advanced options** su 1, **sp_configure**consente di visualizzare un elenco delle opzioni avanzate. Il valore predefinito Ã¨ 0.  
  
 L'impostazione diventa effettiva immediatamente e non richiede il riavvio del server.
