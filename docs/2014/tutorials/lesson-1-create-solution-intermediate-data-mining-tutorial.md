# <a name="lesson-1-creating-the-intermediate-data-mining-solution-intermediate-data-mining-tutorial"></a>Lezione 1: Creazione della soluzione intermedia di data mining (Esercitazione intermedia sul data mining)
  Nell'esercitazione di base sul Data Mining è stato creato un [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] progetto che contiene una soluzione di data mining di dati semplice basato sul nuovo [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)] database.  
  
 Per questa esercitazione si creerà un oggetto separato [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] progetto usando [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]. Si creerà una [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] origine dati che utilizza [!INCLUDE[ssSampleDBDWobject](../includes/sssampledbdwobject-md.md)]e aggiungere alcune nuove viste origine dati per tale origine dati per supportare gli scenari e i tipi di modello.  
  
 In questa lezione è inclusa l'attività seguente:  
  
-   [Creazione di una soluzione e origine dati &#40;esercitazione intermedia sul Data Mining&#41;](../../2014/tutorials/creating-a-solution-and-data-source-intermediate-data-mining-tutorial.md)
