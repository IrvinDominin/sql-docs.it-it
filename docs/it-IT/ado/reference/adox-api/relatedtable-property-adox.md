# <a name="relatedtable-property-adox"></a>Proprietà RelatedTable (ADOX)
Indica il nome della tabella correlata.  
  
## <a name="settings-and-return-values"></a>Le impostazioni e valori restituiti  
 Imposta e restituisce un **stringa** valore.  
  
## <a name="remarks"></a>Osservazioni  
 Il valore predefinito è una stringa vuota ("").  
  
 Se la chiave è una chiave esterna, quindi **RelatedTable** è il nome della tabella che contiene la chiave.
